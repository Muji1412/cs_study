# 🗺️ 비전공자 재취업을 위한 CS 로드맵 (Java 기반)

반가워요! 실무 경험은 있지만 기본기가 부족해서 불안해하는 마음, 충분히 이해합니다.  
하지만 **"경험이 있다"는 건 엄청난 무기**예요.  
CS 지식을 단순히 암기하는 게 아니라, **"아, 그때 그 에러가 이것 때문이었구나!"** 하고 연결하는 순간, 실력은 폭발적으로 성장할 겁니다.

이 로드맵은 **Java 언어**를 메인으로 사용하여, **면접 방어**와 **실무 역량 강화**를 동시에 잡을 수 있도록 설계했습니다.

1.  **자료구조/알고리즘**: 코딩 테스트 통과 및 효율적인 코드 작성 능력 배양
2.  **운영체제/네트워크/DB**: 면접 단골 질문 완벽 대비 및 시스템 동작 원리 이해

## 📅 주차별 커리큘럼 (총 12주 완성 코스)

### [1단계] 기초 체력 다지기 (1~4주차)
이 시기에는 **"직접 구현"**해보는 것이 가장 중요합니다. 눈으로만 보면 절대 내 것이 안 됩니다.

#### 1주차: Java 기초 & 객체지향 (다시 보기)
*   **핵심 키워드:** JVM 메모리 구조(Stack/Heap), GC(Garbage Collection), OOP 4대 특성(캡슐화, 상속, 다형성, 추상화), Interface vs Abstract Class
*   **실습 과제:**
    *   [ ] `ArrayList`를 직접 배열로 구현해보기 (`MyArrayList` 만들기)
    *   [ ] 인터페이스를 활용하여 '결제 시스템'(카드, 현금, 포인트) 설계해보기

#### 2주차: 자료구조 (1) - 선형 구조
*   **핵심 키워드:** Array vs LinkedList, Stack, Queue
*   **실습 과제:**
    *   [ ] Stack을 사용하여 '괄호 짝 맞추기' 문제 풀기
    *   [ ] Queue를 사용하여 '프린터 대기열' 시뮬레이션 구현하기

#### 3주차: 자료구조 (2) - 비선형 구조 & 해시
*   **핵심 키워드:** Tree(Binary Tree), Map(HashMap 동작원리 - 해시 충돌), Set
*   **실습 과제:**
    *   [ ] HashMap의 `put`과 `get` 메서드가 내부적으로 어떻게 동작하는지 그림으로 그려보기 (Hash Bucket, Entry)
    *   [ ] 간단한 주소록 관리 프로그램 만들기 (이름으로 검색 - $O(1)$)

#### 4주차: 알고리즘 기초 (정렬 & 탐색)
*   **핵심 키워드:** 시간 복잡도(Big-O), 정렬(Bubble, Select, Insert vs Merge, Quick), 이진 탐색(Binary Search)
*   **실습 과제:**
    *   [ ] 10만 개의 랜덤 숫자를 만들고, 버블 정렬과 자바 내장 `Arrays.sort()`의 속도 차이 비교해보기
    *   [ ] 이진 탐색으로 '업다운 게임' 봇 만들기

### [2단계] 심화 및 시스템 이해 (5~8주차)
이제 코드를 넘어 **컴퓨터가 어떻게 일하는지**를 배웁니다. 면접에서 가장 많이 털리는(?) 구간이니 집중하세요!

#### 5주차: 운영체제 (OS) - 프로세스와 쓰레드
*   **핵심 키워드:** Process vs Thread, Context Switching, Multi-thread 환경의 동기화 문제 (Synchronized, Deadlock)
*   **학습 포인트:** Java의 쓰레딩 모델과 연결해서 공부하세요.
*   **면접 질문:** "멀티 프로세스와 멀티 쓰레드의 차이는 무엇인가요?"

#### 6주차: 네트워크 (Web의 동작 원리)
*   **핵심 키워드:** OSI 7계층(간략히), TCP/IP(3-way Handshake), HTTP/HTTPS, REST API, GET vs POST
*   **학습 포인트:** 브라우저 주소창에 `google.com`을 쳤을 때 일어나는 일을 설명할 수 있어야 합니다.

#### 7주차: 데이터베이스 (DB) 기초
*   **핵심 키워드:** RDBMS vs NoSQL, Transaction(ACID), Isolation Level, Join 종류
*   **실습 과제:**
    *   [ ] 간단한 게시판 ERD(개체 관계도) 그려보기 (User, Post, Comment)
    *   [ ] Transaction Rollback 상황을 코드로 만들어보기

#### 8주차: 데이터베이스 (DB) 심화 - 인덱스
*   **핵심 키워드:** Index의 원리(B-Tree), Index를 타는 쿼리 vs 안 타는 쿼리, 정규화 vs 반정규화
*   **학습 포인트:** "인덱스를 많이 걸면 무조건 좋은가요?"에 대한 답을 찾아보세요.

### [3단계] 실전 면접 & 아키텍처 (9~12주차)
배운 내용을 종합하고, 실제 면접 상황에 대비합니다.

#### 9주차: Java 심화 (Effective Java 맛보기)
*   **핵심 키워드:** Generic, Stream API, Lambda, Optional, Exception 처리 전략 (Checked vs Unchecked)

#### 10주차: 디자인 패턴
*   **핵심 키워드:** Singleton(주의점), Factory, Strategy, Observer, Adapter
*   **학습 포인트:** 스프링 프레임워크에서 어떤 패턴이 쓰이는지 찾아보세요. (예: DI는 전략 패턴의 응용)

#### 11주차: 시스템 설계 기초 (맛보기)
*   **핵심 키워드:** Load Balancer, Cashing(Redis), Session vs Token(JWT), 동기 vs 비동기 처리
*   **면접 질문:** "사용자가 갑자기 100배로 늘어나면 어떻게 대처하실 건가요?"

#### 12주차: 모의 면접 & 이력서 정리
*   **활동:**
    *   지금까지 공부한 내용을 바탕으로 **"기술 블로그"** 정리하기
    *   예상 질문 50개 리스트 만들고 **육성으로 대답하는 연습** 녹음하기
