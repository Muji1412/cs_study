## 📂 폴더 구조 설명

각 주차별로 폴더가 만들어져 있습니다. 공부한 내용은 해당 주차 폴더에 정리하세요.

*   `Week01_Java_Basics`: Java 기초 및 객체지향 (미션 포함!)

*   `Week03_NonLinear_DS`: 자료구조 2 (트리, 해시)
*   `Week04_Algorithm_Basics`: 알고리즘 등...

### 🏃‍♂️ 바로 시작하는 방법
1.  **`Week01_Java_Basics` 폴더로 이동**하세요.
2.  `mission.md` 파일을 엽니다.
3.  거기에 적힌 **Mission 1: MyArrayList 만들기**부터 시작하세요.
4.  작성한 코드는 그 폴더 안에 `MyArrayList.java` 로 저장하시면 됩니다.
